A. Packing Rectangles

題意

你有$n$個長方形，長度為$h$，寬度是$w$，不可旋轉，找一個最小邊長的正方形容納他們。

解法

對邊長二分搜。設邊長為 $m$，則這個正方，可以容納$(m/w)*(m/h)$個長方形，看它跟 $n$ 的大小，就可以知道 $m$ 接下來的方向。

心得

幫 WA 6次的人QQ。

problem A

B. Ropes

題意

你有 $n$ 條繩子，你要把它們切成 $k$ 段長度一樣的繩子，問最長可以切多少。

解法

對繩子切完後的長度二分搜。然後去看，用這樣的長度可以切出多少條繩子，接著與 $k$ 比大小。值得注意的是本題是浮點數二分搜，詳細的作法可以參考 Sam學長的講義

problem B

C. Very Easy Task

題意

需要複印 $n$ 個文件，但你只有一個文件跟兩台複印機，它們分別要 $x$ 跟 $y$ 的時間來印一個東西，問最少需要的時間。

解法

對時間二分搜。看在 $m$ 時間內可不可以印 $n-1$ 份的文件，最後再加上第一份文件複印所需的時間。Very Easy Task。

problem C

D. Children Holiday

題意

要吹 $m$ 個氣球，有 $i$ 個人幫你吹 有點厲害 ；對於第 $i$ 個人來說，吹一顆氣球要 $t_i$ 分鐘，但吹 $z_i$ 顆後，他就要去休息 $y_i$ 分鐘。大家都超聰明，會照最有效率的方法吹，問其時間跟每個人吹的數量。註: 如果吹完跑去休息，但他之後就不用再吹的話，視作他立刻完成了工作。即休息的時間不會算入

解法

對時間二分搜。看在 $x$ 時最多可以吹幾顆氣球。最後再用答案回推每個人吹的顆數。

心得

這題的小細節很多，是目前我在這邊錯最多次的題目。最後是在學長的幫助下才AC的 QQ。

problem D

E. Equation

題意

求 $x$ 符合 $x^2 + \sqrt{x} = c$。

解法

如果還不夠明顯的話，對 $x$ 二分搜，丟進算式裡。是浮點數二分搜。

心得

hmmmm… 數學。

problem E

F. String Game

題意

現有字串 $t$，然後 $p$ 是 $t$ 的子序列 (subsequence)，接著依著特定的順序開始刪除 $t$ 裡面的字元 (變成 $t’$)，求最多可以做幾步保證 $p$ 仍是 $t$ 的子序列。

解法

對步驟數二分搜。然後把對應的字串生出來，再看 $p$ 是不是依舊在 $t’$ 裡。

心得

我覺得可以有更好的作法，相信正在讀本文的你一定比我聰明QQ。

problem F

G. Student Councils

題意

有 $n$ 個組別，對於第 $i$ 組，有 $a[i]$ 個人。每個學生會要 $k$ 個人，又一個學生會內不能有人來自同一組，問最多能有幾個學生會。

解法

對學生會數 ( 假設為 $m$ ) 二分搜。然後把人塞進去，對於每個組別 $i$，最多只能塞 $m$ 或 $a[i]$ 人到裡面，最後再看人數有沒有達標。

心得

寫這題的時候，一開始想錯，但把思考角度換成，先假設答案再去驗證其合法性的話，答案便呼之欲出。

problem G

H. Hamburgers

題意

Polycarpus 做漢堡，名約「Le Hamburger de Polycarpus」，食譜為一字串，包含三種字元:’B’(bread)、’S’(Sus Sausage)、’C’(Cheese)，現 Polycarpus 身上分別有 $n_b$、$n_s$、$n_c$ 個麵包、香腸跟起司，然後商店裡麵包、香腸跟起司分別賣 $p_b$、$p_s$、$p_c$，有 $r$ 盧比 (俄羅斯的貨幣)，問最多可以做幾個漢堡。

解法

對漢堡數二分搜。算出不夠的材料的價格，然後看買得買不起這樣。

心得

這題算簡單la，但值得注意的是，二分搜的上界不能無腦設 $10^{18}$ 之類的，要大概算一下可能得值。
