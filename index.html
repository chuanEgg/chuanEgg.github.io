第一次架網站

架網站，但為什麼?

沒啥特別的原因，不知道為什麼，突然很想來架部落格。總之，在電Sam學長的幫助下，我也很順利的架起來ㄌ( 拍手 )，也有弄到了自己的domain。反正，就是這樣。嗯…然後週一(後天)要段考，如果考爛了應該會哭，但這個真的蠻有趣的，令人無法自拔。喔對，這邊計畫被用來寫一些文章(心得、筆記或日常廢文之類的)。

架網站，但怎麼弄?

這個我猜沒人在乎。

像我的 blog 也是架在 github 上

Sam Lee

一開始選用 hugo，然而，似乎是命運的安排或其他神奇的理由，我沒辦法正常地 hugo，在經過數個小時的對決之後，我放棄了。然後就轉用 hexo 了。之後乃是十分的順利，過程中只有幾行指令跟小畫家而已。DNS的部分，則是使 Github 給的Education pack裡 namecheap 送的domain，經過掙扎和權衡後，取叫huanc.me。就是這麼的簡單。

架網站，但然後哩?

沒有然後，目前的方針是 “就更阿“。好啦，總之就是把一些值得紀錄的東西放上來拉。希望能透過經營網站，來學一些酷酷的東西，或是用來交女朋友之類的。

結語

我好累喔，第一篇blog居然是廢文，很有我的風格。欸?吐司沒了，我好餓喔QQ。段考也還沒讀，要炸了ㄅ。然而 明天的事就交給明天的我摟。晚安。
